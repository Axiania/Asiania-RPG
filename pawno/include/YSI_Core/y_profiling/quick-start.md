# Quick Start

```pawn
#define RUN_PROFILINGS

#include <a_samp>
#include <YSI_Core\y_profiling>

Profile:LoopV1()
{
	new i = 0;
	while (i != 10)
	{
		++i;
	}
}

Profile:LoopV2()
{
	for (new i = 0; i != 10; ++i)
	{
	}
}

main() {}
```
